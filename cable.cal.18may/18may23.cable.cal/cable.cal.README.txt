Cable Cal 18may23

File naming convention:

<cable ID>.<S-parm>.<load>.<file type>

cable ID:  "UU" is used for the top side of the FE (side A)
           "BOT" is used for the bottom side of the FE (side B)
load:      47 pF capacitor, 66 pF capacitor, 19.46 ohm resistor,
           49.99 ohm resistor, or 194.65 ohm resistor.
file type:   .txt or .s2p  


Measurements under RT conditions, 70F.
